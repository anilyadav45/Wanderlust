🏡 LivoraStays – A Travel Stay Booking Website (Inspired by Airbnb)

LivoraStays is a full-stack web application where users can explore, create, edit, and manage travel stays.
It includes full authentication, cloud image uploads, maps, and a complete CRUD system for listings.

⭐ Features

User Signup, Login & Logout (Authentication with Passport.js)

Cookie + Session management

CRUD operations for Listings (Create, Read, Update, Delete)

MongoDB Atlas (Cloud Database) Integration

Cloudinary for image upload

Mapbox Geocoding API for location-based maps

Reviews system (Add/Delete reviews for stays)

Authorization (Only owner can edit or delete their listings)

🛠️ Tech Stack

Node.js, Express.js

MongoDB Atlas, Mongoose

EJS, Bootstrap

Passport.js, Cloudinary, Mapbox
